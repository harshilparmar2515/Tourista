## 📌 Overview

**Tourista** is a modern travel booking web application built using React and Firebase.
It allows users to explore trips, book travel packages, and manage their bookings securely.

The project demonstrates full-stack development concepts including authentication, database integration, protected routes, and deployment.

---

## ✨ Features

* 🔐 **User Authentication (Google Login)**
* 🧭 **Browse Travel Destinations**
* 📄 **Trip Details Page**
* 📝 **Booking System**
* 📊 **My Bookings Dashboard**
* 🔒 **Protected Routes**
* ☁️ **Realtime Database (Firestore)**
* 📱 **Responsive UI (Mobile + Desktop)**

---

---

## 🧠 How It Works

1. User logs in using Google
2. Browses available trips
3. Selects a trip and fills booking form
4. Booking data is stored in Firestore
5. User can view their bookings in **My Bookings page**

---

## 🔐 Security

* Firebase Authentication ensures secure login
* Firestore Rules restrict data access:

  * Users can only access their own bookings
* Protected routes prevent unauthorized access

## ⚙️ Installation & Setup

```bash
# Clone repository
git clone https://github.com/harshilparmar2515/Tourista.git

# Navigate to project
cd Tourista

# Install dependencies
npm install

# Run development server
npm run dev
```

---

## 🚀 Build & Deploy

```bash
npm run build
```

* Deploy to GitHub Pages
* Deploy to Firebase Hosting
